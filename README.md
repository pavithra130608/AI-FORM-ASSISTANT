# AI Form Assistant

AI Form Assistant is a web application designed to simplify the process of understanding, filling, and managing government forms using intelligent automation techniques.

---

## Features

* AI-based form understanding
* Automated form filling from user input
* Voice input support
* Multi-language assistance
* Upload and process PDF forms
* Data validation
* Eligibility checking for schemes

---

## Tech Stack

* Backend: Python, FastAPI
* Frontend: HTML, CSS, JavaScript
* Libraries: PyPDF2, Pillow (PIL), SpeechRecognition
* Tools: Git, Visual Studio Code

## Installation and Setup

### 1. Clone the repository

```
git clone https://github.com/pavithra130608/AI-FORM-ASSISTANT.git
cd AI-FORM-ASSISTANT
```

### 2. Create a virtual environment

```
python -m venv venv
```

### 3. Activate the environment

```
venv\Scripts\activate
```

### 4. Install dependencies

```
pip install -r requirements.txt
```

### 5. Run the application

```
python app/main.py
```
